# 4. Usar Funcionalidades

> **Guía para descubrir, consultar y ejecutar funcionalidades registradas (`wp_find_feature`, `wp_get_features`).**

---

Una vez que las funcionalidades están registradas, pueden ser recuperadas y ejecutadas usando funciones globales o la instancia del registro.

---

## Encontrar una Funcionalidad Específica (`wp_find_feature`)

Para obtener un solo objeto de funcionalidad por su ID completo (incluyendo el prefijo de tipo):

```php
<?php
/**
 * Encuentra una funcionalidad por su ID.
 *
 * @param string $feature_id El ID completo de la funcionalidad (ej: 'recurso-miplugin/titulo-sitio').
 * @return WP_Feature|null El objeto de funcionalidad si se encuentra y es elegible, null de lo contrario.
 */
function wp_find_feature( string $feature_id ): ?WP_Feature
```
